Predefined variables explanation:
${TYPE}   A PHP class or interface which the plugin observes.
${NAME}   An arbitrary plugin name that identifies a plugin. Also used to merge the configurations for the plugin.
${PLUGIN_TYPE}   The name of a plugin’s class or its virtual type.
${SORT_ORDER}   The order the plugin is executed when there are multiple plugins calling the same adjusted method.
